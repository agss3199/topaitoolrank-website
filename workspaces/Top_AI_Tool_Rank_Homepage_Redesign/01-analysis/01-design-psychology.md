# Design Psychology: What Makes Sites Memorable

## The Memory Problem

Most SaaS homepages follow a predictable formula:
1. Hero with headline + subheadline
2. Floating cards/blobs
3. 2-3 sections (features/services)
4. CTA button
5. Footer

Result: **Zero differentiation.** Visitors forget the site within hours.

## What Creates Recall

### 1. **Novelty Effect** (The Mere Exposure Effect)
- Users remember things they haven't seen before
- Unexpected layouts trigger attention
- Asymmetric designs > symmetric designs (humans expect balance, asymmetry surprises)
- Unfinished or slightly off layouts create cognitive engagement

**Example:** Brutalismo aesthetics use raw typography, irregular grids, and "unpolished" layouts. They feel bold and memorable.

### 2. **Cognitive Load + Delight Balance**
- Too simple = forgettable
- Too complex = frustrating
- Sweet spot = intuitive but not obvious
- Micro-interactions that reward curiosity (hover states, scroll reveals)

**Example:** Figma's landing page uses unusual layout; elements respond to interaction in unexpected ways.

### 3. **Color Confidence**
- Safe colors (blues, grays) are forgettable
- Bold colors are memorable BUT must be used with purpose
- Monochrome with one vibrant accent > pastel palette
- Color conveys emotion (energetic, trustworthy, creative)

**Example:** Stripe uses a bold blue + black. Notion uses simple black + white + blue. Both memorable because of confidence, not complexity.

### 4. **Motion with Intention**
- Purposeless animations are gimmicky
- Meaningful motion guides the eye and tells a story
- Parallax that relates to content (not random)
- Hover states that preview interaction

**Example:** Awwwards-winning sites use scroll-triggered reveals that make sense semantically.

### 5. **Typography as Design**
- Text size hierarchy creates visual rhythm
- Unexpected font pairing creates personality
- Large hero text (2-3x normal) commands attention
- Mixed case, unusual line lengths, or creative text placement

**Example:** Shopify's homepage uses oversized text; letters overflow containers; text becomes graphical.

## Anti-Patterns (What To Avoid)

❌ **Bouncing animations** — cheesy, feels amateur  
❌ **Auto-playing videos** — intrusive, kills Core Web Vitals  
❌ **3D spinning elements** — unless executed with extreme care  
❌ **Too many colors** — looks like a design school experiment  
❌ **Parallax without purpose** — distracting, not engaging  
❌ **Blinking/pulsing** — accessibility issues + annoying  

## SaaS Benchmarks (What's Overdone)

**Overdone:**
- Floating gradient orbs/blobs (every AI tool uses this)
- Particle animations (Canvas-based animations)
- Glassmorphism backgrounds (frosted glass effect)
- Hero with 2-column layout (text left, visual right)
- Smooth scroll with fade-in cards
- Circular avatars with testimonials

**Underused:**
- Bold asymmetric layouts
- Brutalist/raw aesthetic
- Large typography used architecturally
- Isometric or perspective 2.5D designs
- Subversive humor (risky but memorable)
- Monochrome with computed accent color
- Character presence (subtle, not dancing bears)
- White space as design (not just padding)
