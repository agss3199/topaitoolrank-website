## Publishing Workflow

1. **Copy this file**: `cp _template.mdx my-article-slug.mdx`
2. **Edit the frontmatter** — Update all metadata fields. Ensure:
   - `slug` matches the filename (without .mdx)
   - `heroImage` points to a real file in `/public/blog/images/`
   - All tags are lowercase and relevant
   - `status: "published"` to include in the site (use `"draft"` to hide)
3. **Paste article content** — Replace template sections with your ChatGPT output
4. **Add hero image** — Place a 1200×630px image in `/public/blog/images/{slug}.jpg`
5. **Commit and push** — `git add . && git commit -m "feat(blog): add {title}" && git push`
6. **Vercel auto-deploys** — Article goes live in ~90 seconds

## Frontmatter Reference

| Field | Type | Required | Notes |
|---|---|---|---|
